# Code Principles

Examples and demonstrations of programming principles:
- **OOP**: Object-Oriented Programming concepts
- **SOLID**: SOLID principles
- **CLEAN_CODE**: Clean code practices

## Setup

Install dependencies:
```bash
pnpm install
```

## Running Examples

Run any TypeScript file:
```bash
pnpm start <path-to-file.ts>
```

Example:
```bash
pnpm start src/OOP/inheritance.ts
```

For development with auto-reload:
```bash
pnpm dev <path-to-file.ts>
```

Running OOP examples:
```bash
# Core OOP Pillars
pnpm run oop:abstraction
pnpm run oop:encapsulation
pnpm run oop:inheritance
pnpm run oop:polymorphism:notifiable
pnpm run oop:polymorphism:payment-method

# Additional OOP Concepts
pnpm run oop:composition
pnpm run oop:interfaces
pnpm run oop:static
pnpm run oop:getters-setters
```

## Building

Compile all TypeScript files:
```bash
pnpm build
```
